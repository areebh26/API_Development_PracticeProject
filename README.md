# Student Management REST API

A **RESTful API for managing student records** built with **Node.js, Express.js, and MongoDB**.

The API supports full **CRUD operations** and also handles **profile image uploads** using **Multer**.

This project was built to understand how real backend systems are structured and how APIs manage **data, files, and error handling**.

---

## Features

- RESTful API design
- MVC architecture (Models, Controllers, Routes)
- Full CRUD operations for students
- Image upload handling with **Multer**
- Automatic deletion of uploaded images when records are removed
- Proper HTTP status code responses
- API testing using **Postman**

---

## Tech Stack

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### File Handling
- Multer

### Testing
- Postman
